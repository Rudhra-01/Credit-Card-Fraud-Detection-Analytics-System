# Credit Card Fraud Detection & Analytics System

## Overview
Machine Learning project to identify fraudulent credit card transactions and visualize insights using Power BI.

## Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Power BI

## Model Performance
- Accuracy: 99.92%
- Precision: 82%
- Recall: 65%
- F1 Score: 73%

## Features
- Fraud Detection Model
- Streamlit Web App
- Power BI Dashboard
- Exploratory Data Analysis

## Dataset

Kaggle Credit Card Fraud Detection Dataset
